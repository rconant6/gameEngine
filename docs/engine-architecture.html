Macro Overview

Game Loop

The engine runs a fixed update loop. Each frame executes systems in this order:

  1. ActionSys — process input/collision triggers, execute queued actions
  2. PhysicsSys — apply forces, update velocities (stub)
  3. MovementSys — position += velocity * dt
  4. CollisionDetectionSys — detect overlaps, populate collision events
  5. CameraTrackingSys — spring-damper follow target entity
  6. LifetimeSys — decrement timers, mark expired entities with Destroy
  7. RenderSys — draw world-space sprites/text, then screen-space UI

Systems

Pure functions called by the engine each frame. src/systems/

ActionSys
run(world, action_system, trigger_ctx)
1) Process all registered trigger systems (input, collision)
2) Execute queued actions via ActionExecutor
3) Clear queue for next frame
MovementSys

CollisionDetectionSys
run(world, collision_events, debugger)

CameraTrackingSys
run(world, dt)
Query: CameraTracking + ActiveCamera
Immediate mode: snap to target

LifetimeSys
run(world, dt)

Query: Destroy → destroyEntity()
RenderSys

World-space text: Query Transform + Text → drawText
Screen-space pass: Query UIElement + Sprite → anchor-positioned drawGeometry

Scene Format

Custom declarative DSL for defining game scenes. src/scene-format/

Lexer
src: [:0]const u8   index: u32   indent_level: u32
next() → ?Token
Indent-sensitive, emits indent/dedent tokens
Parser
lexer: Lexer   current_tok / previous_tok: Token
parse() → SceneFile
Recursive descent: declarations → components → properties → values
AST
SceneFile { decls: []Declaration }
Declaration = SceneDecl | EntityDecl | AssetDecl | TemplateDecl | ComponentDecl
ComponentDecl = GenericBlock | SpriteBlock{ shape_type } | ColliderBlock{ shape_type }
Property { name, TypeAnnotation{ base_type, is_array }, Value }
Value = number | string | color | bool | vector | assetRef | array
BaseType = vec2 | vec3 | f32 | i32 | u32 | bool | string | color | asset
AssetType = font // + zxl coming

Example

[GameFont:asset font]
  filename:string "orbitron.ttf"

[Player:entity]
  [Transform]
    position:vec2 {0.0, 0.0}
  [Sprite:circle]
    radius:f32 5.0
    fill_color:color #ff0000
  [Tag]
    tags:string "player"
