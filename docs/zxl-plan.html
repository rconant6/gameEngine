Binary Format

HEADER (16 bytes, fixed) [0..3] magic [4]u8 = "ZXL\0" [4] version u8 = 1 [5] flags u8 = 0 (reserved) [6..7] frame_count u16 LE [8..9] canvas_width u16 LE [10..11] canvas_height u16 LE [12] palette_size u8 1-255 (0 = 256) [13..15] reserved [3]u8 = 0 PALETTE (palette_size * 4 bytes) Per entry: [r, g, b, a] index 0 = transparent FRAMES (one per frame, variable size) Per frame: name_len u8 name [name_len]u8 e.g. "walk_01" duration_ms u16 LE 0 = static origin_x i16 LE pivot point origin_y i16 LE reserved u8 = 0 pixels [w*h]u8 palette indices, row-major

64x64, 32 colors, 1 frame ~ 4.25 KB. No compression needed.
